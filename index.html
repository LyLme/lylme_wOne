PHP 版本 必须 ≥ 8.1
MySQL 版本 必须 ≥ 8.0
Web 服务器 推荐 Nginx ≥ 1.18（也支持 Apache / IIS）
PHP 扩展（pdo_mysql） 必须 数据库驱动
PHP 扩展（mbstring） 必须 多字节字符串
PHP 扩展（fileinfo） 必须 文件类型检测
PHP 扩展（openssl） 必须 加密通信
PHP 扩展（gd） 必须 图片处理
PHP 扩展（json） 必须 JSON 处理
PHP 扩展（xml） 必须 XML 解析
PHP 扩展（curl） 必须 HTTP 请求
Redis（可选） 可选 缓存加速（不安装则使用文件缓存）

第一步：设置网站运行目录

⚠ 重要：出于安全考虑，必须将网站运行目录（网站根目录）指向 public 子目录，而不是项目根目录。这样可以防止核心代码、配置文件等被直接访问。
1

添加站点

宝塔面板 → 网站 → 站点设置 → 在"网站目录"一栏中，设置运行目录为 public

# 注意：一定要包含 /public 路径
2

伪静态选择

宝塔面板 → 网站 → 站点设置 → 伪静态 → 选择 thinkphp

1

定位配置文件

找到 Nginx 站点配置文件（通常位于 /etc/nginx/conf.d/ 或 /etc/nginx/sites-available/）

2

设置 root 为 public 目录

# 将 root 指向 public 目录
root /path/to/lylmew/public;

# ❌ 错误示例（不要这样设置）：
# root /path/to/lylmew;
3

重启 Nginx

nginx -t && nginx -s reload
1

修改虚拟主机配置

<VirtualHost *:80>
    DocumentRoot "/path/to/lylmew/public"
    ServerName www.your-domain.com

    <Directory "/path/to/lylmew/public">
        Options -Indexes +FollowSymLinks
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>
2

重启 Apache

sudo systemctl restart apache2
# 或
sudo systemctl restart httpd
1

IIS 中设置物理路径

打开 IIS 管理器 → 右键站点 → 管理网站 → 高级设置 → 物理路径 → 设置为 E:\ltkj\web\public（请根据实际路径修改）

2

IIS 伪静态规则

需要安装 URL Rewrite 模块，然后导入 public/web.config 中的规则（如未提供，可参考下方 Apache 规则自行转换）。

🔗

第二步：配置伪静态规则（URL 重写）

将以下规则添加到 Nginx 配置文件中的 server 块内（ThinkPHP 8 官方规则）：

location / {
    if (!-e $request_filename) {
        rewrite ^(.*)$ /index.php?s=$1 last;
        break;
    }
}

如果是 Apache，public 目录下已自带 .htaccess 文件，无需额外配置。文件内容如下：

<IfModule mod_rewrite.c>
  Options +FollowSymlinks -Multiviews
  RewriteEngine On

  RewriteCond %{REQUEST_FILENAME} !-d
  RewriteCond %{REQUEST_FILENAME} !-f
  RewriteRule ^(.*)$ index.php?s=$1 [QSA,PT,L]
</IfModule>

需要安装 URL Rewrite 模块，然后在 public/web.config 中添加：

<?xml version="1.0" encoding="UTF-8"?>
<configuration>
    <system.webServer>
        <rewrite>
            <rules>
                <rule name="ThinkPHP" stopProcessing="true">
                    <match url="^(.*)$" />
                    <conditions logicalGrouping="MatchAll">
                        <add input="{REQUEST_FILENAME}" matchType="IsFile" negate="true" />
                        <add input="{REQUEST_FILENAME}" matchType="IsDirectory" negate="true" />
                    </conditions>
                    <action type="Rewrite" url="index.php?s={R:1}" />
                </rule>
            </rules>
        </rewrite>
    </system.webServer>
</configuration>
🚀

第三步：运行安装程序

1

访问安装向导

部署完成后，在浏览器中访问：

http://your-domain.com/install.php

系统会自动检测环境、引导填写数据库信息并完成安装。

2

填写数据库信息

按照安装向导提示，填写数据库主机、端口、用户名、密码、数据库名等信息。

如果数据库不存在，系统将自动创建。

3

完成安装

安装成功后，安装程序将自动写入 .env 配置文件，并在项目根目录下创建 install.lock 锁定文件。

4

访问系统

前台首页：http://your-domain.com
后台管理：http://your-domain.com/admin（可在 .env 中修改 ADMIN_PATH 自定义后台路径）

🔑

默认管理员账户

用户名 admin
密码 admin123
⚠ 安全警告：首次登录后请立即修改密码！
🔄

如何重新安装？

如需重新安装系统，请删除install.lock文件后再次访问安装程序： 

rm -f install.lock

然后重新访问 http://your-domain.com/install.php 即可。
